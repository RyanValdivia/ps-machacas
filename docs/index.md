Bienvenido a la página de documentación de nuestro equipo para el proyecto Healthchecks. Healthchecks es una plataforma de monitoreo para tareas programadas (cron jobs) e infraestructuras en segundo plano.

## Objetivos del Proyecto

Nuestro enfoque como equipo de Calidad de Software (QA) se centra en los siguientes hitos:

1. **Garantizar la estabilidad:** Replicar y estructurar las pruebas unitarias y de integración del proyecto base.
2. **Cobertura de Código:** Alcanzar y auditar una cobertura de código (Coverage) mínima del 85%.
3. **Automatización Integral:** Diseñar e implementar flujos de CI/CD utilizando GitHub Actions para automatizar el paso del código desde el Entorno DEV al Entorno QA.
4. **Metodología Ágil:** Gestionar el ciclo de vida del proyecto mediante Sprints, utilizando GitHub Projects y GitHub Issues para las historias de usuario.

## Stack Tecnológico

El entorno tecnológico sobre el cual estaremos realizando las pruebas e integraciones está compuesto por:

* **Lenguaje Core:** Python
* **Framework Web:** Django
* **Base de Datos:** PostgreSQL / SQLite (Entorno DEV)
* **Testing:** Pytest / Coverage.py
* **Gestión y Automatización:** Git, GitHub Actions, GitHub Projects
* **Documentación:** GitHub Pages, Github Wiki
