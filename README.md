# Proyecto Final - Pruebas de Software

Repositorio de trabajo del equipo **LUMINA - Luciernagas** para el curso de **Pruebas de Software** (E.P. Ingenieria de Sistemas, UNSA).

Este proyecto se centra en el aseguramiento de calidad sobre un sistema real basado en Django, aplicando enfoque agil, trazabilidad y automatizacion de pruebas.

## Presentacion del trabajo

- **Proyecto base evaluado:** Healthchecks (`healthchecks/healthchecks`)
- **Objetivo general:** Disenar e implementar una estrategia de QA agil con automatizacion, trazabilidad y gestion colaborativa.
- **Metodologia:** Scrum por sprints.
- **Herramientas principales:** GitHub Projects, GitHub Issues, GitHub Actions, GitHub Pages, Git.
